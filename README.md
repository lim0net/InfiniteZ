# InfiniteZ 🎮
First-Person Shooter Zombie Survival Video Game made for CPS 282 - Multimedia Design<br><br>
**A DEADLY VIRUS HAS WIPED OUT THE HUMAN RACE!** <br><br>
You have been sent back 5 years where it all started. You have 4 days to kill all the zombies <br>and find the antidote before the virus becomes uncontollable. <br>
You are our only hope so don't get touched or the loop will restart! <br>

# Purpose of the Game 🎯
InfiniteZ is a challenging game that challenges the player to complete all 4 levels/waves of zombies without being touched. <br> 
If the player is touched they will be sent back in time sending them back to the beginning of the game to try to complete the mission untouched.
The zombie virus spreads fast so whatever you do...**DON'T LET THEM TOUCH YOU!**
